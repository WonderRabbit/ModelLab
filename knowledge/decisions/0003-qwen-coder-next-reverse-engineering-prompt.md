# 0003 - Qwen-Coder-Next 역공학 프롬프트 추가

## 의사결정
`prompts/user/qwen-coder-next-reverse-engineering.md`를 신규 추가하여,
React/Vertex + Spring + SAP 연동 시스템을 대상으로 역공학 분석 산출물(시퀀스 다이어그램, UI 분석, DFD, 유저 스토리)을 일관 형식으로 생성한다.

## 근거
- 사용자 요청이 단순 코드 생성이 아니라 **아키텍처/데이터 흐름/비즈니스 로직의 역공학 문서화**에 초점을 둠. (`evidence`)
- 기존 `qwen2.5-coder-7b-patterns.md`는 범용 패턴 중심이며, 다중 시스템 연동 역공학용 산출물 계약이 충분히 구체적이지 않음. (`inference`)

## 결과
- 분석 결과물의 재현성 및 비교 가능성이 향상된다.
- 관찰/해석/가설 구분과 근거 ID 강제를 통해 과신 리스크를 낮춘다.
- 후속 평가(playbook 기반)에서 동일 템플릿을 반복 적용할 수 있다.

## 가정 및 한계
- Qwen-Coder-Next의 실제 출력 품질은 아직 저장소 내부 실측으로 검증되지 않았음. (`hypothesis`)
- Vertex AI 연동은 프로젝트별로 존재하지 않을 수 있어 "해당 없음" 분기를 템플릿에 포함함. (`inference`)

## 다음 단계

3. 필요 시 모델별 변형(짧은 출력형/엄격 JSON형)을 추가한다.
